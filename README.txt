## How to Deploy to GitHub + Netlify

### Step 1 — Upload to GitHub
1. Go to your GitHub repo (820-collective-website)
2. Click "uploading an existing file" or drag and drop
3. Upload ALL files including the images/ folder
4. Click "Commit changes"

### Step 2 — Connect Netlify
1. Go to netlify.com and log in
2. Click "Add new site" → "Import an existing project"
3. Choose GitHub → select your 820-collective-website repo
4. Leave all settings as default
5. Click "Deploy site"

### Step 3 — Connect Your Domain
1. In Netlify go to Site Settings → Domain Management
2. Click "Add custom domain"
3. Type: the820co.com
4. Follow the instructions to update your DNS settings
   (usually done in GoDaddy, Namecheap, or wherever you bought the domain)

Your site will be live at the820co.com within minutes to a few hours.
